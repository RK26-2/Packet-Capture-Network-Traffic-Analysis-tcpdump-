# Packet-Capture-Network-Traffic-Analysis-tcpdump-
This project is about 'packet capture and network analysis' using tcpdump in a Linux virtual environment. Acting as a SOC analyst, I captured live 'HTTP' traffic on 'eth0' using '-nn' to prevent DNS leaks and '-c9' to limit packets. I saved the 'pcap' file in the background (&amp;). Analysing via '-r, -v and -X' revealed IP headers and more.
